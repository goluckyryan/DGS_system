1.  Release Notes

Changes in current version:

  • Check for configuration files of the vdct plugins also in sys directory
  • FileChooser on Mac OSX
  • JCA debug plugin future in VDCT
  • DBD path/filename overlaps with the zoom level
  • Print with legend
  • Don't show fields in text area if they are links
  • Copy-paste a field value from a record
  • Search for a record in a record list
  • DrawingSurface canvas size
  • Visibility of the fields
  • Editable combo box
  • Make vdct debug work in latest jca (port to JCA2 and Pure Java CA)
  • Provide 'application runner plugin'
  • Override fields of an already imported record
  • New record dialog focus
  • ... and bugfixes.

Disabilities/missing:

  • include statement is implemented, but saving files containing include statements will also include all the content of includes files.
  • multiple template definitions in one DB are not supported
  • user's manual to be completed

Still there are quite a lot of ideas and nice features to be implemeted in the next releases:

  • fix printing support
  • ...
